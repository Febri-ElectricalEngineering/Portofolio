Recent Projects

Smart Lamp Controller

with HC-SR04

The Smart Lamp Controller with HC-SR04 is an Arduino-based automation system that uses the HC-SR04 ultrasonic sensor to detect the presence of an object within a specific range. If an object (e.g., a hand or a person) is detected within ≤10 cm, the lamp automatically turns ON. When the object moves away, the lamp turns OFF automatically.

IoT-Based Body Temperature and Heart Rate Monitoring

with DHT11 & MAX30100

The Body Temperature and Heart Rate Monitoring System combines the DHT11 and MAX30100 sensors to monitor both body temperature and heart rate. This system uses an Arduino as the central controller, which collects and displays the data from both sensors in real time.

Digital Scoreboard

Using FPGA Altera DE1 Cyclone II

As part of my FPGA project, I designed and implemented a digital scoreboard using the Altera DE1 Cyclone II FPGA This scoreboard can be used for various sports or games, displaying the score in real-time with user-controlled inputs.
